CSV Data Analysis with Pandas

A Python-based data analysis project that demonstrates how to read, clean, and analyze CSV data using the Pandas library. This project was developed as Task 2 of my Python Internship.


 Project Overview

This project loads employee data from a CSV file, performs data cleaning by handling missing values and removing duplicate records, and generates basic statistical insights. It demonstrates the practical use of the Pandas library for data preprocessing and analysis.


 Features

- Read employee data from a CSV file
- Display the original dataset
- Handle missing values using the column mean
- Remove duplicate records
- Display the cleaned dataset
- Calculate the average employee salary
- Simple and efficient implementation using Pandas



Technologies Used

- Python 3.x
- Pandas
- CSV Files

How to Run

Clone the Repository

```bash
git clone https://github.com/K-Apoorva-reddy/Python-data-handling.git
```

Navigate to the Project

```bash
cd Python-data-handling
```

Install Pandas

```bash
pip install pandas
```

Run the Program

```bash
python main.py
```
